### 🎲 Rodando o projeto na sua maquina

```bash
# Clone este repositório
$ git clone https://github.com/cardoso0/pokedex

# Não esqueça de navegar até a pasta
$ cd pokedex

# Instale as dependências
$ npm install

# Execute a aplicação
$ npm run dev

```
